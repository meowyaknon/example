Description:
เขียนโปรแกรม
เพื่อรับลิสต์ของตัวเลขแล้วคืนลิสต์ใหม่ที่เรียงจากน้อยไปมาก
ห้ามใช้ sorted() หรือ list.sort()

Input Specification:
- อินพุตเป็นลิสต์ของจำนวนเต็ม
- ความยาวของลิสต์ ≥ 1

Output Specification:
- คืนลิสต์ของตัวเลขเรียงจากน้อยไปมาก

Example Testcases:
Input: [5, 1, 4, 2, 3] → Output: [1, 2, 3, 4, 5]
Input: [10, -2, 7, 0] → Output: [-2, 0, 7, 10]
Input: [3, 3, 1] → Output: [1, 3, 3]
