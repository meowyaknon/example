Description:
เขียนโปรแกรม เพื่อหาผลรวมของเลขคู่ทั้งหมดในลิสต์ที่รับเข้ามา
ถ้าไม่มีเลขคู่ ให้ผลลัพธ์เป็น 0

Input Specification:
- อินพุตเป็นลิสต์ของจำนวนเต็ม
- ความยาวของลิสต์ ≥ 1

Output Specification:
- คืนค่าจำนวนเต็ม เป็นผลรวมของเลขคู่ทั้งหมดในลิสต์

Example Testcases:
Input: [1, 2, 3, 4, 5, 6] → Output: 12
Input: [7, 9, 11] → Output: 0
Input: [10, -2, 8] → Output: 16
